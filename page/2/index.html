Eclipse跨版本升级（大版本升级）

 

  在2011年下半年的时候，由于某些机缘巧合，致使我初次接触了"Eclipse+ADT"进行 Android 的相关学习与研究，当时还没有 Android Studio，那时候 Android 集成开发环境也只有用"Eclipse+ADT"，之后，那是我第一次用 Eclipse，之前一直在做的工作是PC平台反病毒的研究与MFC程序的编写，IDE也只是 Visual Studio 用的多，更如井底之蛙一般的还对 Java 抱有一丝抵触情绪。
  Eclipse 可以通过增加各类插件来支持不同种类的开发工作，例如 Pydev 用来做 Python 开发，ADT 用来做 Android 开发，等等。
  Eclipse 的自动升级功能，一般只会在相应的大版本下进行小版本更新，如果希望用上最新版本的 Eclipse ，也就是进行大版本更新，我们应该怎么办做呢？
