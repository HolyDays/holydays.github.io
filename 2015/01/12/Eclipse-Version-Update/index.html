Eclipse跨版本升级（大版本升级）

 

  在2011年下半年的时候，由于某些机缘巧合，致使我初次接触了"Eclipse+ADT"进行 Android 的相关学习与研究，当时还没有 Android Studio，那时候 Android 集成开发环境也只有用"Eclipse+ADT"，之后，那是我第一次用 Eclipse，之前一直在做的工作是PC平台反病毒的研究与MFC程序的编写，IDE也只是 Visual Studio 用的多，更如井底之蛙一般的还对 Java 抱有一丝抵触情绪。
  Eclipse 可以通过增加各类插件来支持不同种类的开发工作，例如 Pydev 用来做 Python 开发，ADT 用来做 Android 开发，等等。
  Eclipse 的自动升级功能，一般只会在相应的大版本下进行小版本更新，如果希望用上最新版本的 Eclipse ，也就是进行大版本更新，我们应该怎么办做呢？

难道..要重新下载新版，再重新安装一遍全部插件吗？

如果你遇到了类似问题，请按下面步骤试试：

1、打开 Menu -> Help -> Install New Software…

2、打开"Available Software Sites"，会发现有一条当前版本的升级检测地址，
  例如目前我电脑中的版本为"Juno"(v4.2)，对应地址为"http://download.eclipse.org/releases/juno"

3、现在我们打开"https://eclipse.org/downloads"
  看一下"Eclipse"的最新版本：为"Luna"(v4.4.1)(2015年1月12日数据)

4、点击"Add…"按钮，添加一条:
  "Name：Luna"
  "Location：http://download.eclipse.org/releases/luna"

Tips:
  只需修改Location为形如:"http://download.eclipse.org/releases/[LastVersionName]"即可。
   (即替换 [LastVersionName] 为所需升级的大版本名)

5、之后，我们打开 Menu -> Help -> Check for Updates，

6、经过检测后，可以看到，找到了4.4.1的版本更新。

7、更新后，经测试一切正常。

恭喜，你学会了Eclipse大版本更新。（全文完）
